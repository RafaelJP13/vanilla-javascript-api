# Products API

Products API enables product management, built using Node.js, 

#

## Installation

```bash
  npm install
  npm start
```
    
## API Reference

#### Create Product

```http
  POST /api/products
```

#### Get all Products

```http
  GET /api/products/
```

#### Get Product

```http
  GET /api/products/:id
```

#### Edit Product

```http
  PUT /api/products/:id
```

#### Delete Product

```http
  DELETE /api/products/:id
```
