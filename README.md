# 📦 Products API

A simple RESTful Products API built with **Node.js** and **Vanilla JavaScript** for managing product data through standard CRUD operations.

## ✨ Features

- Create products
- Retrieve all products
- Retrieve a product by ID
- Update existing products
- Delete products
- RESTful API architecture
- Lightweight and easy to understand

---

## 🚀 Getting Started

### Installation

Clone the repository:

```bash
git clone https://github.com/RafaelJP13/vanilla-javascript-api.git
```

Navigate to the project folder:

```bash
cd vanilla-javascript-api
```

Install dependencies:

```bash
npm install
```

Start the server:

```bash
npm start
```

---

## 📡 API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| `POST` | `/api/products` | Create a new product |
| `GET` | `/api/products` | Retrieve all products |
| `GET` | `/api/products/:id` | Retrieve a product by ID |
| `PUT` | `/api/products/:id` | Update a product |
| `DELETE` | `/api/products/:id` | Delete a product |
